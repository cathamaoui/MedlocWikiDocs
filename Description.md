# 📝 Project Description

Medloc is a specialized staffing platform built to solve the nursing shortage in Moncton, New Brunswick. By using automated matching algorithms and SMS integration, it connects clinics with available nurses in real-time. 

Our goal is to ensure that no healthcare shift goes unfilled, providing better care for the community and flexible opportunities for healthcare professionals.
