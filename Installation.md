# 🛠 Installation Guide

Follow these steps to set up the Medloc development environment on your local machine.

### 1. Prerequisites
* **Node.js** (v16.0 or higher)
* **Git** installed on your system
* A **Stripe Account** for payment processing integration

### 2. Clone the Repository
Open your terminal and run the following command:
```bash
git clone https://github.com
```
