
# 🛠 Medloc Technical Stack

This document outlines the core technologies powering the Medloc platform.

## 📱 Frontend (Web & Mobile)
- **React Native:** For a cross-platform mobile experience (iOS/Android).
- **React.js:** For the hospital/employer dashboard.

## ⚙️ Backend & API
- **Node.js with Express:** High-concurrency server to handle thousands of users.
- **Twilio API:** 💬 Automating SMS shift offers and notifications.

## 🗄 Database & Security
- **PostgreSQL:** Reliable relational data for shifts and payroll.
- **HIPAA Compliant Hosting:** (e.g., AWS or Azure) To ensure medical data privacy.

## 💸 Payroll Integration
- **Stripe Connect:** To automate 24-hour payouts to nurses.
